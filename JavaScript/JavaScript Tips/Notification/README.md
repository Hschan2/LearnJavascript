# 알람 (Notification)
웹 사이트에서 자바스크립트로 알람을 주거나 받을 수 있는 방법을 발견합니다.   

알람을 Push하거나 혹은 Toast로 받는 방법을 찾고 파악합니다. 그리고 이 후 웹 사이트 개발에 활용할 계획을 갖습니다.
